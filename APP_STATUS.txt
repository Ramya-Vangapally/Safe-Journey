❌ SMS NOT SENDING?
   Solution:
   1. Set emergency contact in preferences
   2. Verify phone in Twilio dashboard
   3. Check backend logs for specific error
   4. Restart backend if needed

❌ APP NOT LOADING?
   Solution:
   1. Make sure frontend is running: http://localhost:5173
   2. Make sure backend is running: http://127.0.0.1:8000
   3. Check for CORS errors in browser console
   4. Try clearing browser cache

❌ DATABASE ERRORS?
   Solution:
   1. Database is fresh (just recreated)
   2. Tables created automatically on startup
   3. If errors persist, delete safejourney.db and restart
